Room数据库使用指北

发表于 2017-12-25 | 分类于 Android

Room依赖

1.创建一个数据库表

比如我想创建一个文章的表

1.@Entity(tableName = “articles”)
创建一个表,名字叫做”articles”

2.@PrimaryKey()
主键,每个表都需要
@PrimaryKey(autoGenerate = true)
表示,这个_id是自增的.

3.@ColumnInfo(name = “articleUrl”)
private String url;
默认名字为 url
但是可以使用ColumnInfo(name = “”)指定一个名字

2.数据库的初始化

1.extends RoomDatabase 必须继承RoomDatabase

2.@Database(entities = {ArticleModel.class},version = 1,exportSchema = false)

  • entities 包含的表(当前只有一个,ArticleModel,如果有多个加入即可,”,”隔开)
  • version = 1 数据库的版本
